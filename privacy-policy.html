Privacy Policy for Ninja ReCaptions

Last Updated: October 15, 2024

Summary: Ninja ReCaptions respects your privacy. We do not collect, store, or share any personal data. All translation processing happens between your browser and Google Translate API.

1. Information Collection and Use

Ninja ReCaptions does NOT collect, store, or transmit any personal information or user data to our servers. We do not have servers that store user data.

2. Data Processing

The extension processes the following data locally in your browser:

  • Caption Text: Text from Loom video captions is temporarily processed and sent to Google Translate API for translation purposes only
  • User Preferences: Your language selection and toggle settings are stored locally in your browser using Chrome's storage API
  • Translation Cache: Translated text is cached locally in your browser to improve performance and reduce API calls

3. Third-Party Services

Ninja ReCaptions uses Google Translate API to provide translation services:

  • Caption text is sent to Google Translate API for translation only
  • No personal identifying information is sent with translation requests
  • Google Translate's privacy policy applies to data sent to their service: https://policies.google.com/privacy

4. Data Storage

All data is stored locally on your device:

  • Local Storage Only: Extension settings and cached translations are stored in your browser's local storage
  • No Cloud Storage: We do not store any data on external servers or cloud services
  • User Control: You can clear all stored data at any time by removing the extension or clearing browser data

5. Permissions

The extension requires the following permissions:

  • storage: To save your language preferences and translation cache locally in your browser
  • activeTab: To access the current Loom video page and read captions for translation
  • host_permissions (loom.com): To inject the translation overlay into Loom video pages
  • host_permissions (translate.googleapis.com): To access Google Translate API for translation services

6. Data Security

  • All data processing happens in your browser
  • We do not have access to your data
  • No data is transmitted to our servers (we don't have any)
  • Translation requests to Google Translate API are made over secure HTTPS connections

7. Analytics and Tracking

We do NOT use any analytics, tracking, or monitoring tools.

  • No usage statistics are collected
  • No user behavior is tracked
  • No cookies are set by this extension
  • No third-party analytics services are used

8. Children's Privacy

Our extension does not knowingly collect any information from anyone, including children under 13. The extension does not collect personal data from any users of any age.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. We encourage you to review this Privacy Policy periodically.

10. Your Rights and Data Control

You have complete control over your data:

  • All data is stored locally on your device
  • You can disable the extension at any time
  • You can remove the extension to delete all stored data
  • You can clear browser data to remove cached translations

11. Compliance

This extension is designed to be compliant with:

  • Chrome Web Store Developer Program Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)

12. Contact Us

If you have any questions or concerns about this Privacy Policy or the Ninja ReCaptions extension, please contact us:

  • GitHub: https://github.com/rental-ninja/ninja-recaptions
